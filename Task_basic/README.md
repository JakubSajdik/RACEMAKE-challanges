# Task — Basic Product Engineer Challenge

## Run

```bash
cd Task_basic
bun run challenge.ts
```

If you don’t have Bun:

```bash
npx tsx challenge.ts
```

You should see per-lap PitGPT-style output for both stint laps, a stint summary, then the four validation checks with ✅ if everything lines up.

## Files

| File | Purpose |
|------|---------|
| `challenge.ts` | Full solution: types, data, analysis, coach, runner + validation. |
| `WRITEUP.md` | How I approached Level 1–3 (human-readable). |

Level 3 is answered only in `WRITEUP.md` (not duplicated as a giant comment in the code).
